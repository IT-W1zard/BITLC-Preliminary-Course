# Variables, Data Types, Operators

# Variables

Overview

Variables are used to store data in a program.
They allow you to save values and reuse them later.

You will learn:

- What variables are
- How to create variables
- How to assign values

## 1. What is a Variable?
A variable is a container for storing data.
Each variable has:

- A name
- A value

## 2. Creating Variables

You create a variable by assigning a value to it.

Example:
```
SET age = 18
SET name = "Alex"
```
## 3. Using Variables
Variables can be used in calculations or outputs.
Example:
```
SET a = 5
SET b = 3
SET sum = a + b
OUTPUT sum
```
## Summary
Variables help you:
- Store data
- Reuse values
- Make programs flexible

# Data Types

Overview

Data types define what kind of data a variable can store.
You will learn:
- Different data types
- When to use them

## 1. Common Data Types
- Integer → whole numbers (e.g. 5, -2)
- Float → decimal numbers (e.g. 3.14)
- String → text (e.g. "Hello")
- Boolean → true or false

## 2. Example
```
SET number = 10
SET price = 9.99
SET text = "Hello"
SET isValid = true
```
## 3. Why Data Types Matter
Data types control:
- What values are allowed
- What operations can be performed

Summary
- Data types help you:
- Organize data
- Prevent errors
- Use correct operations

# Operators
Overview

Operators are used to perform actions on variables and values.

You will learn:
- Arithmetic operators
- Comparison operators
- Logical operators

## 1. Arithmetic Operators
- addition
- subtraction
- multiplication
- / division
- 
Example:
```
SET a = 5
SET b = 3
SET result = a + b
```
## 2. Comparison Operators
- = → equal
- != → not equal

- x > greater than

- X < less than

- >= greater or equal

- <= → less or equal

Example:
```
IF a > b THEN
    OUTPUT "a is greater"
END IF
```
## 3. Logical Operators
AND
OR
NOT
Example:
```
IF age > 18 AND isStudent = true THEN
    OUTPUT "Eligible"
END IF
```
## Summary

Operators allow you to:

Perform calculations
Compare values
Combine conditions
