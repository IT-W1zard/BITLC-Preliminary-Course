# Lists and Arrays

Overview

Lists and arrays are used to store multiple values in a single variable.  
They allow you to work with collections of data efficiently.  

You will learn:

- What lists and arrays are
- How to create them
- How to access elements
- How to use them in programs

## 1. What are Lists and Arrays?

Lists and arrays are data structures that store multiple values.

Instead of using many variables, you can store all values in one structure.

## 2. Creating a List / Array

You can store multiple values in a single variable.

Example:
```
SET numbers = [1, 2, 3, 4, 5]
SET names = ["Alex", "Sam", "Chris"]
```
## 3. Accessing Elements

Each element has a position (index).

Index usually starts at 0.

Example:
```
OUTPUT numbers[0]
OUTPUT names[1]
```
## 4. Modifying Elements

You can change values inside a list or array.

Example:
```
numbers[2] = 10
```
## 5. Using Loops with Lists

Lists are often used together with loops.

Example:
```
FOR i = 0 TO 4 DO
    OUTPUT numbers[i]
END FOR
```
## 6. Adding Elements

You can add new elements to a list.

Example:
```
ADD 6 TO numbers
```
## 7. Summary

Lists and arrays allow you to:

- Store multiple values
- Access data using indices
- Process data efficiently with loops

They are essential for handling larger amounts of data in programs.
