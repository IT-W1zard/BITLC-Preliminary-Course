# Data Input and Output

Overview

Data input and output are essential parts of any program.
They allow a program to receive data from the user and display results.

You will learn:

- What input and output are
- How to read data
- How to display results
- How input and output are used in programs

## 1. What is Input?

Input is data that a program receives.

It can come from:

- The user (keyboard)
- Files
- Other programs

## 2. What is Output?

Output is data that a program sends out.

It can be:

- Text on the screen
- Results of calculations
- Messages to the user

## 3. Basic Input and Output

Example:
```
INPUT name
OUTPUT "Hello " + name
```
## 4. Input with Processing

Programs usually combine input, processing, and output.

Example:
```
INPUT A
INPUT B
SUM = A + B
OUTPUT SUM
```
## 5. Multiple Inputs and Outputs

A program can handle multiple values.

Example:
```
INPUT firstName
INPUT lastName
OUTPUT "Name: " + firstName + " " + lastName
```
## 6. User Interaction

Input and output allow interaction between user and program.

Example:
```
OUTPUT "Enter your age:"
INPUT age
OUTPUT "You are " + age + " years old"
```
## 7. Summary

Data input and output allow programs to:

- Receive data
- Process information
- Display results

They are necessary for making programs interactive and useful.
