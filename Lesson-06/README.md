# Simple Data Processing

Overview

Simple data processing involves taking input, processing it, and producing an output.
It is one of the most basic concepts in programming.

You will learn:
- The input → process → output principle
- How to structure simple programs
- How to perform basic calculations

## 1. Input → Process → Output
Every simple program follows this structure:
- Input → data is received
- Process → data is calculated or changed
- Output → result is displayed

## 2. Example: Addition Program
```
INPUT A
INPUT B
SUM = A + B
OUTPUT SUM
```
## 3. Example: Calculate Area
```
INPUT length
INPUT width
AREA = length * width
OUTPUT AREA
```
## 4. Example: Average of Numbers
```
INPUT a
INPUT b
INPUT c
AVERAGE = (a + b + c) / 3
OUTPUT AVERAGE
```
## 5. Steps in Data Processing
Step 1: Read input
Step 2: Process data (calculate)
Step 3: Output result

## 6. Summary
Simple data processing is based on:
- Input → Process → Output
- Clear step-by-step logic
- Basic calculations
It is the foundation for all more complex programs.
