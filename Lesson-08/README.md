# Writing Small Programs

Overview

In this lesson, you apply everything you have learned to write small programs.
You combine variables, control structures, and logic to solve simple problems.

You will learn:

- How to build small programs step by step
- How to use Turtle for simple graphics
- How to create a basic calculator
- How to combine logic and visualization

## 1. What are Small Programs?

Small programs are simple applications that solve a specific task.

They usually follow:

- Input → Process → Output
- Clear structure
- Simple logic

## 2. Example: Simple Calculator

A calculator program performs basic operations.
```
INPUT A
INPUT B
INPUT operator

IF operator = "+" THEN
    RESULT = A + B
ELSE
    IF operator = "-" THEN
        RESULT = A - B
    END IF
END IF

OUTPUT RESULT
```
## 3. Using Turtle (Graphics)

Turtle is used to draw shapes and create simple visual programs.

Example: Drawing a Square
```
REPEAT 4 TIMES
    MOVE FORWARD 100
    TURN RIGHT 90
END REPEAT
```
Example: Drawing a Circle
```
REPEAT 36 TIMES
    MOVE FORWARD 10
    TURN RIGHT 10
END REPEAT
```
