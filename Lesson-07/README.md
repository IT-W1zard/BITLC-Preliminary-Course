# Control Structures: IF, FOR, WHILE

Overview

Control structures determine how a program flows.
They allow you to make decisions and repeat actions.

You will learn:
- What control structures are
- How to use IF statements
- How to use FOR loops
- How to use WHILE loops

## 1. What are Control Structures?
Control structures control the order in which instructions are executed.

There are three main types:
- Selection (IF)
- Iteration (FOR, WHILE)

## 2. IF Statement (Selection)

The IF statement is used to make decisions.

Example:
```
INPUT number
IF number > 0 THEN
    OUTPUT "Positive"
ELSE
    OUTPUT "Not Positive"
END IF
```
## 3. FOR Loop (Fixed Repetition)

A FOR loop is used when the number of repetitions is known.

Example:
```
FOR i = 1 TO 5 DO
    OUTPUT i
END FOR
```
## 4. WHILE Loop (Condition-Based Repetition)

A WHILE loop repeats as long as a condition is true.

Example:
```
SET counter = 1
WHILE counter <= 5 DO
    OUTPUT counter
    counter = counter + 1
END WHILE
```
## 5. Comparison of Loops
- FOR → used when the number of iterations is known
- WHILE → used when the condition controls repetition

## 6. Steps in Control Structures

Step 1: Define condition or range  
Step 2: Execute instructions  
Step 3: Repeat or decide based on condition  

## 7. Summary

Control structures allow you to:

- Control program flow
- Make decisions (IF)
- Repeat actions (FOR, WHILE)

They are essential for writing efficient and logical programs.
