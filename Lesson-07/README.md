# Control Structures: IF, FOR, WHILE

Overview

Control structures determine how a program flows.
They allow you to make decisions and repeat actions.

You will learn:
- What control structures are
- How to use IF statements
- How to use FOR loops
- How to use WHILE loops

## 1. What are Control Structures?
Control structures control the order in which instructions are executed.

There are three main types:
- Selection (IF)
- Iteration (FOR, WHILE)
